# 📝 Pagina Blanca

Una página extremadamente simple para escribir notas directamente en el navegador.

Sin cuentas.  
Sin distracciones.  
Sin interfaces innecesarias.

---

## 🚀 ¿Por qué existe?

Este proyecto nació por cansancio.

Cansancio de usar aplicaciones de notas llenas de cosas que no necesito:

- registros
- sincronización obligatoria
- interfaces pesadas
- tiempos de carga innecesarios

Y también por algo más práctico:

Muchas veces solo quiero escribir algo rápido mientras estoy desarrollando…  
y termino abriendo la consola del navegador para usar `console.log` o escribir texto temporal.

Esto reemplaza ese hábito con algo mucho más limpio.

---

## ✨ Qué hace

- Pantalla completamente en blanco
- Escribes como si fuera un documento
- Guarda automáticamente en el navegador (`localStorage`)
- No necesitas guardar manualmente
- No hay botones, ni UI visible

---

## 🧠 Filosofía

Esto no es una app de notas.

Es un espacio vacío para pensar.

---

## ⚙️ Cómo usar

1. Abre la página
2. Escribe
3. Cierra
4. Vuelve cuando quieras → tu texto sigue ahí
